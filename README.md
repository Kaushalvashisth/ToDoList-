# To-Do-List WebApp
It is ToDoList WebApp which allows you to add items in multiple list(ex:-Shoping List, Work List etc).

![Overview](/Screenshots/img.gif)

## Tech used
+ MongoDB 
+ Node.js
+ Express.js
+ Embedded JavaScript (EJS)
+ HTML5, CSS  and Bootstrap 4

## Get Started
+ `Download the project`
+ `npm install`
